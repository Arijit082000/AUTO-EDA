# BI Software Automatic EDA

## Project Overview
This project performs **Automatic Exploratory Data Analysis (EDA)** on the **BI Software Recommendation Dataset** using **Python** and **ydata-profiling**.

## Features
- Load dataset with Pandas
- Generate an automatic EDA report
- Detect missing values
- Analyze distributions
- Correlation analysis
- Data type summary

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- ydata-profiling

## Project Structure
```
BI_Software_Automatic_EDA/
│── BI_Software_Autometic_EDA.ipynb
│── BI_Software_recommendation_dataset.csv
│── BI_Software_EDA_Report.html
│── requirements.txt
└── README.md
```

## Installation
```bash
pip install -r requirements.txt
```

## Run
Open the notebook and execute all cells.

## Output
The notebook generates a comprehensive HTML EDA report.
